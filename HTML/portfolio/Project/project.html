Farm Invoice Manager


Farm Invoice Manager is a desktop-based poultry farm billing management system developed using C# with database integration. It helps manage users, products, customers, and sales efficiently through a user-friendly interface. The system allows invoice generation, sales tracking, and data validation for accurate operations. This project simplifies daily farm management tasks and improves overall productivity.



Flap Fury


Flap Fury is a 2D computer graphics based game developed using C++ with OpenGL and GLUT. The game features interactive gameplay where players control a flying character to avoid obstacles and score points. It includes multiple levels, animated environments, and a dynamic day-night transition system for better user experience. This project demonstrates core computer graphics concepts like animation, transformation, and collision detection in a practical way.
